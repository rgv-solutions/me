Mobirise

Mobirise Website Builder is an application that allows users to create and publish bootstrap websites, without coding. I have created several extensions for people to use - for free.

Disclaimer:
Download and install at your own risk. Always make sure you have proper backups of your projects. If you experience issues, uninstall the extension and see if the issues are then gone, before contacting Mobirise support.

How to install

  1. Download the extensions you want to have (these are .mbrext files)
  2. Start Mobirise
  3. Click Extensions in the menu on the left
  4. Click Import at the top of the screen and select the file you just downloaded
  5. Confirm installing by clicking import anyway (this is important)
  6. The extension should now be installed

Custom HTML v0.3

Mobirise doesn't come with a block that allows you to insert plain HTML into your pages. There's a paid extension that can handle that (and much more!), but if you're on a budget and just want to insert some custom HTML, this extension allows you to do just that. Drag the "custom html" block into your website, just like you do with any other block. Then change the contents using the gear icon.

This extension can only be used with Bootstrap themes, on both Windows and Mac.

Mailform v1 (beta)

With this extension you can easily add independent contact forms to your website that don't depend on Formoid. The required PHP is generated from within Mobirise. All you have to do is change some settings and you're good to go.

Enable the mailform under "Site Settings", you can also configure the global settings there. You can even create a mail template and style it using HTML:



Form specific settings can be defined in the form itself. Simply click on one of the fields and the following window will appear. You can change the subject of the e-mail as well as customize the error message, in case something goes wrong.



This extension can only be used with Bootstrap themes, on both Windows and Mac.

Note: in order for the mail form to work, your web server needs to support PHP.

White Label v3

By default Mobirise inserts meta tags into your websites containing text and links to Mobirise. This extension hides these tags, in case you don't want these. You can enable this setting under "Site Settings".

This extension can be used with Bootstrap and AMP themes, on both Windows and Mac. You will also need Mobirise 4.10.17 or higher.

Code Editor v0.4 (beta)

With this extension, you can edit the HTML and CSS of any block (M4 or AMP).

This is not the official Code Editor by Mobirise. Instead, this Code Editor is entirely FREE of charge, but has less features than the official one.
